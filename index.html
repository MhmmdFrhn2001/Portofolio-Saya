Aplikasi Penyimpanan Rak Buku

Dalam aplikasi ini, user dapat melakukan input data seperti input judul buku, input nama penulis, input tahun terbit, tambah buku, dan mencari data buku. Apabila buku sudah selesai dibaca maka akan berpindah ke halaman yang sudah selesai dibaca, jika buku belum selesai dibaca maka buku akan dikembalikan ke halaman yang belum selesai dibaca. Selain itu data buku dapat dihapus apabila buku sudah selesai dibaca maupun belum selesai dibaca. Dalam aplikasi ini menggunakan LocalStorage, jika aplikasi ditutup data tersebut tidak akan hilang dan akan tersimpan di lokal, kecuali data tersebut dihapus secara permanen. Aplikasi ini menggunakan HTML, CSS, dan JavaScript sebagai logika dalam menjalankan aplikasi ini.
